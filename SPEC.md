# BeatForge - 智能鼓机伴奏软件

## 1. 概念与愿景

**BeatForge** 是一款专业级智能鼓机伴奏软件，融合传统硬件鼓机的操作体验与现代 AI 辅助编曲能力。它既是音乐人的节奏工作站，也是初学者的练习伙伴——通过直观的鼓组可视化、丰富的节奏库和智能鼓点生成，让打击乐变得触手可及且充满乐趣。

核心体验：**"所见即所打，所想即所得"** — 每个鼓部件都有对应的视觉反馈，调节参数时声音变化即时可闻。

## 2. 设计语言

### 美学方向
参考 Roland TR-808/909 的复古科技感 + 现代音乐制作软件（如 Ableton、FL Studio）的深色专业风格。荧光色点缀在深色背景上，营造出舞台灯光般的沉浸感。

- 基础单位: 8px
- 鼓组垫片: 4x3 网格布局，每个垫片 80x80px
- 间距: 16px (组件内), 24px (组件间), 32px (区块间)
- 圆角: 8px (小元素), 16px (卡片), 24px (面板)

- **响应感**: 点击反馈 50ms，粒子爆发 200ms

- **节奏同步**: 可视化元素与节拍同步脉动
- **物理模拟**: 鼓槌落下有重力感，弹起有弹性

## 4. 功能与交互

### 核心功能

#### 4.1 鼓组控制
- **8个可分配鼓垫**: 默认 Kick/Snare/HiHat/CloseHH/OpenHH/Tom1/Tom2/Crash
- **点击触发**: 触摸/点击即发声，支持鼠标/触控/ MIDI 输入
- **力度感应**: 虚拟力度条显示击打力度 (0-127)
- **长按编辑**: 长按 500ms 进入该鼓垫的编辑模式
- **双击录制**: 双击进入该鼓的步进录制

#### 4.2 节奏引擎
- **BPM 范围**: 40-240，支持 Tap Tempo
- **拍号**: 4/4, 3/4, 6/8, 5/4, 7/8
- **步进音序器**: 16/32/64 步可选
- **Swing/Shuffle**: 0-100% 可调
- **预置节奏库**: 
  - 流行 (50+): 经典四拍、Trap、Hip-Hop、放克、迪斯科
  - 摇滚 (40+): 经典摇滚、英伦、车库、朋克、金属
  - 电子 (60+): House、Techno、Trance、Dubstep、Drum & Bass
  - 爵士 (30+): Swing、Bebop、Bossa、Fusion
  - 拉丁 (40+): 桑巴、伦巴、恰恰、曼波、雷鬼

#### 4.3 智能节奏生成
- **风格选择**: 基于音乐风格的 AI 生成
- **参数调节**:
  - 复杂度 (1-5): 音符密度
  - 密度 (1-5): 填充频率
  - 动态 (1-5): 力度变化幅度
  - 可预测性 (1-5): 重复与变化比例
- **生成模式**:
  - 完整曲目 (intro/verse/chorus/bridge/outro)
  - 单段落 (4/8/16 小节)
  - 无限模式 (永不重复)
- **一键优化**: 根据现有模式智能填充空缺部分

#### 4.4 混音控制
- **独立通道**: 每个鼓部件独立音量 (0-100%)
- **声像调节**: 左/中/右 定位
- **静音/独奏**: 快速 A/B 测试
- **效果器**:
  - 压缩器 (Threshold/Ratio/Attack/Release)
  - EQ (Low/Mid/High)
  - Reverb (Room/Hall/Plate)
  - Delay (Tempo Sync 选项)
  - Bit Crusher (8/12/16 bit)

#### 4.5 可视化系统
- **波形显示**: 实时音频输出波形
- **频谱分析**: 32-band FFT 频谱
- **节拍灯**: 16 个 LED 跟随节拍
- **MIDI 信号**: 鼓垫按下时的力度可视化
- **历史记录**: 最近 10 秒的波形回溯

#### 4.6 音序器
- **绘制模式**: 点击步进格打开/关闭
- **绘制工具**: 画笔、橡皮擦、线条
- **复制粘贴**: 片段复制、跨音轨复制
- **量化**: 自动对齐到网格
- **力度曲线**: 每步独立力度值

#### 4.7 文件管理
- **保存格式**: .bfproj (BeatForge Project)
- **导出格式**: WAV, MP3, MIDI
- **云同步**: 保存到用户云端
- **节奏包**: 导入/导出自定义节奏库

#### 4.8 MIDI/音频接口
- **MIDI 输入**: 识别外部 MIDI 控制器
- **MIDI 输出**: 导出为标准 MIDI 文件
- **ASIO 支持**: Windows 低延迟音频驱动
- **Core Audio**: macOS 原生音频支持
- **外接接口**: 自动识别 USB 音频接口

### 交互细节

#### 鼓垫交互
| 动作 | 响应 |

## 5. 组件清单

### 5.1 DrumPad (鼓垫)
- **默认**: 深灰背景 + 白色图标 + 底部标签
- **悬停**: 边框高亮 + 轻微上浮阴影
- **按下**: 缩小至 95% + 颜色填充 + 粒子爆发
- **静音**: 对角线 + 灰色调
- **独奏**: 外圈金色边框
- **录制**: 红色脉冲边框

### 5.2 StepSequencer (步进音序器)
- **网格**: 4 行 × 16 列
- **关闭**: 深灰格
- **开启**: 对应鼓垫颜色
- **当前步**: 白色竖线指示
- **播放头**: 发光竖线扫过
- **选中**: 橙色边框

### 5.3 TransportBar (传输控制栏)
- **播放按钮**: 圆形，霓虹绿填充
- **暂停按钮**: 圆形，霓虹黄填充
- **停止按钮**: 方形，霓虹红填充
- **BPM 显示**: 数字 + 旋钮双模式

### 5.4 MixerChannel (混音通道)
- **推子**: 垂直滑块，100px 高
- **电平表**: 双通道 LED 指示
- **旋钮**: 圆形编码器 (声像/EQ)
- **按钮**: 静音 (M) / 独奏 (S)

### 5.5 PatternSelector (节奏选择器)
- **缩略图**: 鼓组示意小图标
- **标签**: 节奏名称 + 风格标签
- **预览**: 悬停时快速试听
- **收藏**: 星标置顶

### 5.6 WaveformDisplay (波形显示器)
- **画布**: 黑色背景 + 霓虹绿波形
- **刻度**: 时间轴 (秒) + 电平轴 (dB)
- **光标**: 播放位置指示

### 数据模型

```typescript
interface DrumMachine {
  id: string;
  name: string;
  bpm: number;
  swing: number;
  timeSignature: [number, number];
  patterns: Pattern[];
  currentPatternId: string;
  mixer: MixerState;
}

interface Pattern {
  id: string;
  name: string;
  style: string;
  steps: number; // 16/32/64
  tracks: Track[];
}

interface Track {
  drum: DrumType;
  steps: Step[];
  velocity: number[];
  mute: boolean;
  solo: boolean;
  volume: number;
  pan: number;
}

interface Step {
  active: boolean;
  velocity: number;
  probability: number; // 0-100, 触发概率
}

interface MixerState {
  channels: Channel[];
  master: Channel;
  effects: Effect[];
}

interface Channel {
  id: string;
  volume: number; // 0-1
  pan: number; // -1 to 1
  mute: boolean;
  solo: boolean;
  effects: EffectInstance[];
}
```

### 存储方案
- **本地**: IndexedDB (浏览器) / SQLite (桌面)
- **云端**: Firebase Firestore 或自建 API
- **导出**: JSON (项目文件), WAV/MP3 (音频), MIDI (音符数据)

### 性能优化
- **音频**: 使用 AudioWorklet 实现低延迟 (<10ms)
- **渲染**: Canvas 离屏渲染 + requestAnimationFrame
- **内存**: 采样池复用，避免重复加载
- **网络**: Service Worker 缓存静态资源
